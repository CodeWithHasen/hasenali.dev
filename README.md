# 🌐 Hasen Ali – Full Stack Developer Portfolio

A modern, responsive, and SEO-optimized portfolio website built using **React.js**, **React Router**, **JavaScript**, **Bootstrap 5**, and **CSS3**.

This portfolio showcases my technical skills, professional projects, education, experience, and contact information through a clean, responsive, and recruiter-friendly user interface.
